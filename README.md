# AIML 2013: Introduction to Computer Vision
Spring 2026 | Rose State College

This repository contains my labs, notebooks, and projects
for AIML 2013: Introduction to Computer Vision.

## Contents

- **Lab 1: Edge Detection & Blurring Kernels**  
  Applies spatial filters to images using kernels to smooth noise and highlight edges. Introduces images as numerical arrays and demonstrates how matrix operations reveal structural features. Explores how filtering transformations influence machine perception.

- **Lab 2: How Machines Read (TF-IDF & HOG)**  
  Implements parallel pipelines for text and images by converting raw data into feature vectors using TF-IDF and HOG. Compares retrieval performance and examines strengths and limitations of handcrafted feature extraction methods. Highlights shared constraints across language and vision tasks.

- **Lab 3: How Machines Represent Text and Images**  
  Replaces handcrafted features with learned embeddings from neural networks for both text and images. Constructs vector-based retrieval systems and compares similarity across modalities. Examines how different data types are represented in dense vector spaces and where the analogy diverges.
