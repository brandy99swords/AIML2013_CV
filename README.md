# AIML 2013: Introduction to Computer Vision
Spring 2026 | Rose State College

This repository contains my labs, notebooks, and projects
for AIML 2013: Introduction to Computer Vision.

## Contents
- **Lab 1: Edge Detection & Blurring Kernels**  
  Apply spatial filters to images using kernels to smooth noise and highlight edges. This lab introduces how images are processed as numerical arrays and how simple matrix operations reveal structure. You’ll experiment with filters and reflect on how these transformations shape machine perception.

- **Lab 2: How Machines Read (TF-IDF & HOG)**  
  Build parallel pipelines for text and images by converting raw data into feature vectors using TF-IDF and HOG. You’ll compare how each method retrieves similar items and analyze where these handcrafted approaches succeed and fail. The goal is to uncover the shared limitations of feature extraction across modalities.

- **Lab 3: How Machines Represent Text and Images**  
  Replace handcrafted features with learned embeddings from neural networks for both text and images. You’ll generate vector representations, build retrieval systems, and compare how similarity works across modalities. This lab highlights how different data types can be unified through dense vector spaces—and where that analogy breaks down.
